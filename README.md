# Vue
Vue Web Presentation
# 技术栈
node（express）+vue（element ui）
## Project setup

```
npm install / yarn
```

### Compiles and hot-reloads for development

```
npm/yarn run serve
```

### Compiles and minifies for production

```
npm/yarn run build
```

### Lints and fixes files

```
npm/yarn run lint
```

### set up nodejs
node/nodemon server
