# Vue Web Presentation

# technology stack
node（express）+vue（element ui）+mongodb
## node_modules setup

```
npm install / yarn
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```

### set up nodejs
```
node/nodemon server
```

cnpm / yarn / pnpm if slow
